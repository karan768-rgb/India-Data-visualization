# India Data Visualization Dashboard

An interactive data visualization dashboard built using Streamlit and Plotly to explore district-level data across India.

---

## Live Demo
[![Open in Streamlit](https://img.shields.io/badge/Open%20in-Streamlit-red?logo=streamlit)](https://india-data-insights.streamlit.app)

---

## About the Project

This project enables users to explore multiple district-level parameters such as population, literacy rate, sex ratio, and more using an interactive map.

Users can:
- Select a state or view overall India
- Choose primary and secondary parameters
- Analyze data through size and color encoding

---

## Features

- Interactive map visualization using Plotly Mapbox
- Dynamic filtering by state
- Multiple parameter selection
- Size and color-based comparison
- Real-time updates based on user input
